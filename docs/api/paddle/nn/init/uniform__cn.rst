.. _cn_api_paddle_nn_init_uniform_:

uniform\_
-------------------------------

.. py:class:: paddle.nn.init.uniform_(tensor, a=0.0, b=1.0)

将输入张量的值设置为均匀分布的随机数，该操作会直接修改输入张量。

参数
::::::::::::

    - **tensor** (Tensor) - 输入张量。
    - **a** (float) - 均匀分布的下限，默认值为 0.0。
    - **b** (float) - 均匀分布的上限，默认值为 1.0。
