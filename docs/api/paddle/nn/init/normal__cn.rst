.. _cn_api_paddle_nn_init_normal_:

normal\_
-------------------------------

.. py:class:: paddle.nn.init.normal_(tensor, mean=0.0, std=1.0)

将输入张量的值设置为正态分布的随机数，该操作会直接修改输入张量。

参数
::::::::::::

    - **tensor** (Tensor) - 输入张量。
    - **mean** (float) - 正态分布的均值，默认值为 0.0。
    - **std** (float) - 正态分布的标准差，默认值为 1.0。
