Size
-------------------------------

.. py:class:: paddle.Size()
``paddle.Tensor.size()`` 的返回类型，用于描述张量的维度大小。作为 ``tuple`` 的子类，支持所有常见的序列操作（如索引、切片、拼接等）。

**参数**
:::::::::
    - ``*args``：表示维度的整数序列或多个整数参数。

**返回**
:::::::::
    - ``Size``：表示张量维度的特殊元组子类。
